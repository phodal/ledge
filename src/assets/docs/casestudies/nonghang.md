# 农业银行

来源《[大型国有金融企业研发中心 DevOps 规划与实践](https://www.infoq.cn/article/493NdQc44E07TV2w1is5)》

## 建设思路
 
  - 规范
     - 建立统一质量视图
     - 打造 DevOps 组织规范
  - 流程
     - 构建持续交付流水线
     - 推进自动化测试
     - 完善运营监控
  - 工具
     - 建设 DevOps 集成平台
